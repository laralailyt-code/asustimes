┌─ 風險指數速查 ────────────────────────────────────────────────────────────┐
│                                                                             │
│ 🔴 指數 ≥60   → 高風險 ⚠️  立即關注，考慮備用方案                         │
│ 🟡 指數 28-59 → 中風險 📌 持續監控，準備應急預案                          │
│ 🟢 指數 <28   → 低風險 ✅ 正常運營                                         │
│                                                                             │
│ 供應商顏色對照：                                                           │
│ 🟢 綠色 = 安全無風險                                                       │
│ 🟡 黃色 = 監控中                                                           │
│ 🔴 紅色 = 高風險（可能停產、延遲）                                        │
│                                                                             │

┌─ 原物料價格解讀 ──────────────────────────────────────────────────────────┐
│                                                                             │
│ 曲線在【歷史高點】 → 目前成本高，等待下跌機會                             │
│ 曲線在【歷史低點】 → 目前成本低，適合採購或備貨                           │
│ 曲線在【中間】     → 價格適中，正常狀態                                   │
│ 曲線【平穩】       → 價格穩定，成本可預測                                 │
│ 曲線【上升趨勢】   → 短期成本上漲，影響利潤                               │
│ 曲線【下降趨勢】   → 短期成本下跌，利潤機會                               │
│                                                                             │
