## 平台概述

**ASUSTIMES** 是一個為華碩及科技產業工作者設計的**即時情報監控平台**，提供：
- 🔴 **即時新聞聚合** — 來自8個主流新聞源的科技產業新聞
- 📊 **原物料與匯率行情** — 實時追蹤關鍵材料和匯率走勢
- 🌐 **供應鏈風險評估** — 地緣政治、氣候、罷工等風險監控

**更新方式**：自動每30分鐘刷新新聞 | 每日08:00/12:00/15:00更新行情 | 實時監控風險事件

### 使用行情頁面


#### 2️⃣ 選擇原物料

- 每條線代表歷史價格走勢（通常過去1年的日線數據）

#### 3️⃣ 查看圖表細節
- 將滑鼠懸停在曲線上，顯示該日期的具體價格

- 點擊來源連結可檢視更詳細的官方數據

### 圖表解讀
```
📍 最高點/最低點 → 判斷目前價位是否便宜
📈 上升趨勢 → 短期成本上升壓力
📉 下降趨勢 → 短期成本下降機會
平台期 → 價格穩定，成本可預測
```

## 🌐 供應鏈風險監控

### 功能介紹
通過**地圖**和**數據**監控全球供應鏈風險，包括：
- 🌊 **颱風/洪水** — 港口、運輸中斷
- 🔴 **地震** — 工廠損傷、停產
- ⚔️ **地緣政治衝突** — 台灣海峽、中東、烏克蘭
- ✊ **罷工** — 台積電、三星等關鍵廠商
- ⚡ **停電** — 限電風險

### 3個子功能模組

#### 📍 事件地圖 (Events & Map)
**左上面板**：實時列出當前風險事件

| 事件圖示 | 含義 | 顯示條件 |
|--------|------|---------|
| 🌊 | 颱風/洪水 | **近3天內**（超過3天不顯示） |
| 🔴 地圖中 | 地震（M≥5.0） | 持續顯示，但不影響風險分數 |
| ⚔️ | 地緣政治衝突 | 近21天新聞（台灣海峽、中東等） |
| ✊ | 罷工 | 僅限**晶圓廠**相關（TSMC、Samsung等） |
| ⚡ | 停電/限電 | 關鍵字偵測 |

**事件嚴重程度**：
- 🔴 **CRITICAL** = 最嚴重（如強颱、大地震、戰爭）
- 🟠 **HIGH** = 高風險（如颱風級、M6+地震）
- 🟡 **MED** = 中等風險
- 🟢 **LOW** = 輕微風險

**週期篩選**：
```
近一週 / 近二週 / 近四週 / 近八週
→ 篩選不同時間範圍的事件
```

> ⚠️ **重要**：颱風/洪水**僅顯示3天內**的事件。超過3天自動隱藏。

#### 🗺️ 供應商分布地圖 (Suppliers Map)
**右側地圖**：顯示華碩供應商地理分布

| 標記顏色 | 含義 |
|---------|------|
| 🟢 **綠色** | 該地區安全，無風險 |
| 🟡 **黃色** | 監控中（有單個事件或多個低風險事件） |
| 🔴 **紅色** | 高風險（多個重大事件同時發生） |

**解讀地圖**：
- 供應商密集地區為較大圓點（如台灣、日本、中國、東南亞）
- 地圖自動顯示距供應商500km範圍內的風險事件

#### 📊 風險指數 (Risk Index)
**下方面板**：按地區統計風險等級

| 指數等級 | 分數範圍 | 燈號 | 含義 |
|---------|---------|------|------|
| ⚠️ **高** | ≥60 | 🔴 | 立即關注，風險高 |
| 🟡 **中** | ≥28 | 🟡 | 持續監控 |
| 🟢 **低** | <28 | 🟢 | 正常運營 |

**風險指數計算邏輯**：
```
若該地區有華碩供應商：
  風險 = 供應商狀態(50%) + 周邊事件(30%) + 地區整體事件(20%)

若該地區無供應商（如中東、非洲）：
  風險 = 地區事件嚴重度 × 0.85
  （仍影響航運路線、能源、零件轉口）
```

**關鍵事件**（新功能）：
- 掃描近3天新聞，檢測「台積電/三星/凱俠 + 停工」關鍵字
- 自動將該廠商所在地300km內的供應商標為🔴高風險

### 如何使用

#### 步驟1：查看事件列表
1. 點擊上方「近一週/近二週/近四週」切換時間範圍
2. 左上面板列出該週期內的所有風險事件
3. 每個事件顯示：日期、事件類型、位置、嚴重程度

#### 步驟2：在地圖上定位
1. 點擊事件名稱，地圖自動定位到該位置
2. 鼠標懸停供應商圓點，顯示該地區的企業數量和風險狀態
3. 地圖下方顯示該地區的詳細風險指數

#### 步驟3：檢查供應商影響
1. 點擊「供應商分布」標籤，切換到供應商地圖
2. 查看華碩供應鏈在該地區的分布
3. 綠/黃/紅圓點顏色表示風險等級

#### 步驟4：解讀風險指數
1. 點擊「風險指數」標籤，查看地區匯總
2. 紅燈 = 立即採取措施（如尋找備用供應商）
3. 黃燈 = 加強監控，準備應急預案

### 數據解讀技巧

**原物料價格走勢**：
- 紅色漲→綠色跌→紅色漲 = 波動性高（成本難預測）
- 平穩曲線 = 價格穩定（成本可控）

- 無AI摘要 = 該分類新聞不足或無重大突發

## 常見問題 (FAQ)

### Q: 為什麼某個原物料沒有更新？
**A**: 可能原因：
1. 來源網站維護（如sci99.com更新）
2. 數據延遲（LME數據有時差）
3. 系統自動使用上次已知價格（仍會記錄）

### Q: 颱風為什麼消失了？
**A**: 颱風/洪水**僅顯示3天內的事件**。超過3天自動隱藏，因為影響已過。

### Q: AI摘要為什麼有時沒有？
**A**: 
1. 該分類新聞太少（少於2篇）
2. AI服務暫時不可用（自動降級為標題+片段）
3. 新分類首次載入（等待背景處理）

### Q: 供應商地圖為什麼看不到具體廠商名字？
**A**: 設計上只顯示**聚合風險指數**，保護供應商隱私。若需具體廠商信息，請聯絡業務單位。

### Q: 我可以自訂監控的商品嗎？
**A**: 目前不支持自訂。若需監控特定商品，請反饋至開發團隊。

### Q: 數據可以匯出嗎？
**A**: 目前不支持直接匯出。建議：
1. 原物料 = 直接到來源網站（LME、sci99.com）
2. 新聞 = 點擊標題進入原網站複製
3. 風險數據 = 截圖或聯絡IT
